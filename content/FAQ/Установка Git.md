---
title: Как скачать Git?
---

**Git** — это движок, который позволяет Obsidian отправлять твои заметки в облако (GitHub) и забирать их оттуда. Без него плагин Obsidian Git работать не будет.

#### 1. Установка

Перейди на [git-scm.com](https://git-scm.com/), скачай инсталлятор и нажимай «Next» во всех окнах. Настройки по умолчанию оптимальны.

#### 2. Предварительная настройка (Identity)

После установки Git нужно «представиться», чтобы в истории изменений было видно, кто внес правки. Открой Git Bash на Windows, для этого кликни пкм на рабочем столе и выбери **Open Git Bash here**. В него введи две команды:

Bash

```
git config --global user.name "Твое Имя или Ник"
git config --global user.email "your@email.com"
```

> [!IMPORTANT] Email должен совпадать с тем, на который зарегистрирован твой аккаунт на GitHub.

#### 3. Проверка связи

Введите в терминале: `git --version`
![[Pasted image 20260102014758.png]]

Если в ответ вы увидели номер версии (например, `git version 2.4x.x`), значит всё готово. Теперь можно открывать Obsidian и настраивать плагин синхронизации.

---

### Совет для упрощения (Мастерский лайфхак)

Если ты не хочешь каждый раз вводить пароль при синхронизации через Obsidian, при установке на Windows убедись, что выбрана опция **«Git Credential Manager»**. Это позволит тебе один раз залогиниться через браузер, и дальше всё будет летать на автомате.
